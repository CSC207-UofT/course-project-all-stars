Class name: Hospital
Parent class: Command

Responsibilities:
* Stores the list of patients in the hospital
* Stores the list of doctors working in the hospital
* Stores the number of rooms in the hospital, the year when it was founded, the name and the priority protocol of the hospital
* Can admit and discharge patients
* Can hire doctors
* Can return details regarding the hospital, containing the name, year of foundation, number of patients being treated

Collaborators:
* Patient
* Doctor
* ArrayList
